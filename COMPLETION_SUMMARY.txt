## 🔍 KEY IMPLEMENTATION DETAILS

### Fine Calculation (Core Feature)
```php
Fine = ₱10 × Overdue Days × Outstanding Books
Example: 5 days overdue, 2 books outstanding = ₱100
```

**Real-time preview in return form using JavaScript**
- User changes return date → Fine updates instantly
- User changes quantity → Fine recalculates
- Shows breakdown of calculation

### Partial Returns
```
Example: Student borrows 3 books, returns 2
- 1 book still outstanding
- Fine calculates for 1 outstanding book
- Available count updated correctly
```

### Availability Tracking
```
Available Count = Total Quantity - Current Borrowed Quantity
Automatically updated when books are borrowed/returned
Prevents overbooking
```
